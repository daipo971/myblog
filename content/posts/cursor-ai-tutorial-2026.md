## 核心技巧

### 1. Cmd+K — 内联编辑

选中代码，按 `Cmd+K`，输入你要改的内容。比如：

```
把这段代码改成 TypeScript，加上类型定义
给这个函数添加错误处理
重构这个组件，拆成更小的函数
```

### 2. Cmd+L — AI 对话

按 `Cmd+L` 打开对话面板，可以问任何关于项目的问题：

```
这个项目的架构是什么样的？
这个函数的时间复杂度是多少？
哪里可能出 bug？
```

### 3. @ 语法 — 引用文件

在对话中用 `@` 引用特定文件：

```
帮我看看 @utils/helpers.ts 这个文件，优化一下性能
```

### 4. Composer (Cmd+I) — 多文件编辑

这是 Cursor 最强大的功能。一次对话可以修改多个文件：

```
帮我创建一个用户登录功能，包括 API 路由、前端页面和数据库模型
```
