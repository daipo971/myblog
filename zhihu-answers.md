## 问题：AI模型量化有哪些实用技术？

### 回答：

AI模型量化是部署优化的关键技术，我总结了以下实用技术：

### 📊 量化技术详解

**1. 量化类型**
- 动态量化：运行时量化，适应性强
- 静态量化：训练后量化，性能更好
- 混合量化：不同层不同精度

**2. 量化方法**
- 均匀量化：简单，计算快
- 非均匀量化：精度高，计算复杂
- 对称量化：实现简单
- 非对称量化：精度更高

**3. 量化级别**
- 8-bit：平衡精度和性能
- 4-bit：极致压缩，精度损失可控
- 2-bit：极端压缩，需要特殊处理

### 🔧 主流框架支持

**PyTorch量化：**
```python
# 动态量化
model_quantized = quantize_dynamic(model, {torch.nn.Linear}, torch.qint8)

# 静态量化
model_prepared = prepare_fx(model, {'': default_qconfig})
model_quantized = convert_fx(model_prepared)
```

**TensorFlow Lite量化：**
```python
# 转换为TFLite模型
converter = tf.lite.TFLiteConverter.from_keras_model(model)
converter.optimizations = [tf.lite.Optimize.DEFAULT]
tflite_model = converter.convert()
```

**ONNX Runtime量化：**
```python
# 动态量化
quantize_dynamic("model.onnx", "model_quantized.onnx", weight_type=ort.quantization.QuantType.QUInt8)
```

### 📈 量化效果对比

| 量化级别 | 模型大小 | 推理速度 | 精度损失 |
|----------|----------|----------|----------|
| FP32 | 100% | 1x | 0% |
| INT8 | 25% | 2-4x | <1% |
| INT4 | 12.5% | 3-6x | <3% |
| INT2 | 6.25% | 4-8x | <5% |

**量化技巧：**
- 校准数据集：使用代表性数据
- 逐通道量化：CNN模型效果好
